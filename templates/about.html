About This App

This is a simple task management web application built with Flask.

Features:

  • Add new tasks with titles and descriptions
  • Mark tasks as completed or reopen them
  • Delete tasks you no longer need
  • View all tasks with their creation dates
  • RESTful API endpoint for task data

Technology Stack:

  • Backend: Python Flask
  • Frontend: HTML, CSS, JavaScript
  • Templating: Jinja2

This is a demonstration application. In a production environment, you would want to use a proper database instead of in-memory storage.
